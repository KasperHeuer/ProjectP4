# ProjectP4

## Installatie via git clone

Volg deze stappen om het Laravel-project lokaal te starten:

1. Clone de repository.
2. Ga naar de projectmap `Echo`.
3. Installeer de PHP-afhankelijkheden met `composer install`.
4. Kopieer `.env.example` naar `.env` en stel je databasegegevens in.
5. Genereer een applicatiesleutel met `php artisan key:generate`.
6. Voer de migraties uit met `php artisan migrate`.
7. Installeer de frontend-afhankelijkheden met `npm install`.
8. Start de ontwikkelserver met `composer run dev`.

Daarna kun je de applicatie openen in je browser via de URL die Laravel toont.
